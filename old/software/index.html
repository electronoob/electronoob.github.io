Open-source software

asc (JavaScript)
Assembly scripting language, with some inline C-like syntax - the aim is to be small yet moderately useful where a processor restricts the end-user from modifying code at runtime. A lightweight virtual machine running on the microcontroller parses bytecode generated by Asc to provide the option of modifying the application feature-set at run-time.
beverly-crusher (C)
Audio crushing tool generates 1-bit PCM output with playback routines for both Arduino and AVR-C. Supports ALSA for preview. Hackaday article.
buildincrement (C)
fork of: http://buildnumber.sourceforge.net/
mubox (Arduino)
Emulating vt100 terminal for a telnet-like login with shell command access over a serial connection. This project has stalled due to it's dependency on Asc. To provide commandline tools and a dynamic configuration mubox depends on the Asc bytecode parser which is currently under development
